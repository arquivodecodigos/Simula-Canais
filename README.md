DESCRIÇÃO DETALHADA:

A hidráulica computacional oferece soluções eficientes para problemas de escoamento em canais livres, permitindo cálculos precisos com recursos acessíveis. Este trabalho apresenta um simulador web desenvolvido em HTML5 e JavaScript para estimar a profundidade da lâmina d’água (y) em seções trapezoidais, utilizando a Equação de Manning e o Método da Bisseção. O algoritmo foi estruturado com critérios claros de convergência, definição de intervalo inicial e tolerância numérica, garantindo simplicidade e reprodutibilidade. A proposta destaca-se pela integração entre rigor técnico e praticidade, sendo aplicável em ensino, pré-dimensionamentos e apoio à tomada de decisão em projetos hidráulicos. Além disso, o estudo aponta caminhos para evoluções futuras, como a inclusão de rugosidade variável, escoamentos não uniformes e ampliação para diferentes geometrias.
