SIMULAÇÃO COMPUTACIONAL DA LÂMINA D'ÁGUA EM CANAIS LIVRES

Código completo para a Simulação Computacional da Lâmina d'Água em Canais Livres desenvolvido em HTML + CSS + JavaScript para o nosso TCC (Trabalho de Conclusão de Curso) em Engenharia Civil na PUC-GO no 2º semestre de 2025.

O método computacional usado para aproximar o valor de y (algura da lâmina d'água) foi o Método da Bisseção.
